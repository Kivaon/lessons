🎧 Как включить приятный мужской голос для озвучки уроков?

По умолчанию ваша система использует стандартный женский голос. Чтобы переключить озвучку на мужской баритон, выполните простую разовую настройку для вашего устройства:

Apple (Mac / iPhone / iPad)

  1. Откройте Системные настройки → Универсальный доступ → Устный контент.
  2. В пункте Системный голос выберите Управление голосами...
  3. Найдите Русский и скачайте голос Yuri (Юрий). Рекомендуется версия «Улучшенный».
  4. Обновите эту страницу.

Windows 10 / 11

  1. Откройте Параметры (Win + I) → Время и язык → Распознавание речи.
  2. В разделе Голоса убедитесь, что установлен пакет для русского языка.
  3. Выберите в качестве стандартного голоса Microsoft Pavel (Павел).
  4. Перезапустите браузер и обновите страницу.

Android (Смартфоны)

  1. Зайдите в Настройки → Расширенные настройки (или Доступность) → Язык и ввод → Синтез речи.
  2. Убедитесь, что выбран Синтезатор речи Google. Нажмите на иконку шестерёнки рядом с ним.
  3. Выберите Установка голосовых данных → Русский.
  4. Выберите мужской вариант (обычно это Голос №2, №3 или голос с маркировкой male).

После выполнения настройки просто обновите эту страницу, и сайт автоматически переключится на новый голос.
